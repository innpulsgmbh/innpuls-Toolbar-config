# Datenschutzerklärung - Chrome Extension

## Datenerhebung
Diese Chrome Extension erhebt folgende Daten:
- Name und E-Mail-Adresse über Microsoft Entra ID SSO
- OAuth-Authentifizierungstoken (lokal gespeichert)

## Datenverwendung
- Authentifizierung und Benutzeranmeldung
- Anzeige von Benutzerinformationen in der Extension
- Keine Weitergabe an Dritte

## Datenspeicherung
- Daten werden lokal im Browser gespeichert
- Keine Übertragung an externe Server
- Token werden automatisch nach Ablauf gelöscht
